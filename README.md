## `Observer Interesction API`

사용자의 **Viewport**에 따라서, 요소들의 이벤트를 **비동기적**으로 설정이 가능

## `useRef Hooks`

React 이용시,Hooks을 통해 **`DOM`** 구조에 접근이 가능
